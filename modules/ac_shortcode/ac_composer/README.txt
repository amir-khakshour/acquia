README.txt
==========

A module for editing node/block content visually. This module works with built-in 
and defined shortcodes within DiamondLayers   shortcode module.


INSTALLATION
======================
1- Enable 'DiamondLayers Visual Composer' and required modules from admin section: admin/modules
2- Enable 'jQuery Update' module
3- Download jQuery UI 1.8.24 from jQuery UI repository: https://code.google.com/p/jquery-ui/ .
	 Extract the downloaded archive file and replace 'external', 'themes' and 'ui' folders inside 
	 of extracted folder with the old ones into 'jquery_update/replace' folder.
4- Have fun!
